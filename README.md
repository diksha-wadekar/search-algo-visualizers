# DSA Mini Project: Linear Search & Binary Search Visualizer

An interactive visualizer built to demonstrate how Linear Search and Binary Search work step by step. Users can input a custom array and a target key, then watch the algorithm search through the array with live index tracking and step traces.

## Features
- Custom comma-separated array input
- Custom search key input
- Step-by-step animated visualization with adjustable speed
- Live step trace log as the search runs
- Displays time and space complexity for each algorithm
  - Linear Search: Time O(n), Space O(1)
  - Binary Search: Time O(log n), Space O(1)
- Binary Search auto-sorts the input array before searching (with on-screen note)
- Dark themed, responsive UI

## Tech Stack
- HTML5
- CSS3
- JavaScript (array traversal logic, DOM updates, animation timing)

## How to Run

2. Open `index.html` in any browser
